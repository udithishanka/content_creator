# Content Creator

Prototype CrewAI project that orchestrates a multi-agent content production pipeline.

## Workflow Steps
1. Trend Detection
2. Planning
3. Research & Fact-Checking
4. Outline
5. Draft
6. Edit & SEO
7. Media Generation
8. Assemble Outputs
9. Compliance & Brand QA
10. Publish/Schedule
11. Analytics Feedback

Run the pipeline (requires valid LLM credentials):

```bash
python pipeline.py
```
